Data

The data files are below. Within the archive is a README.md file that contains a more detailed description of that data, and a short python script to load it. A shorter description is as follows. Each downloadable archive contains a CSV file for each institution. The file for each institution is a csv with columns for each data set, with one column per year of data. The first line of each file is the years, and the anonymized donations are sorted.
